# Agentic Playground

Next.js single-page experience wrapped around the word “Лол”. The page offers a looping emoji animation and a playful counter to keep the energy light.

## Getting Started

```bash
npm install
npm run dev
```

Visit `http://localhost:3000` once the dev server boots.

## Scripts

- `npm run dev` – Start the Next.js dev server
- `npm run build` – Create a production build
- `npm run start` – Serve the production build
- `npm run lint` – Run ESLint using Next.js defaults

## Tech

- Next.js App Router
- React 18
- CSS animations with custom gradients
